# kb（知识库问答）

**核心能力：** RAG（检索增强生成）问答。接收用户问题，检索知识库文档，结合 LLM 生成答案。

## 触发关键词

当用户消息包含以下关键词时，OpenClaw 将自动调用此 Skill：
- 通用疑问：知识库、查询、怎么用、如何、是什么、多少
- 业务操作：规则、流程、接口、配置、开户、销户、划转、转账
- 故障排除：问题、错误
- 交易相关：融资、融券、开仓、平仓、交易、委托、下单
- 账户相关：账户、账号、席位、保证金、资金、现金、股份、持仓、市值
- 风险管理：征信、利息、负债、合约、净资产、授信、额度、风控、预警、平仓线、警戒线、维持担保比例
- 系统名称：低延时、顶点、撮合、symbol、agw、柜台、节点、席位号

## 工作流程

```
用户问题 → 关键词匹配触发 → 知识库搜索 → 文档内容获取 → LLM 生成答案 → 返回答案+来源
```

## 安装

```bash
# 复制到 OpenClaw 服务器
scp -r lark-kb-qa user@server:~/.openclaw/skills/

# 确保设置环境变量
export MINIMAX_API_KEY="your-api-key"
```

## 使用方法

用户直接提问即可自动触发，例如：
- "低延时撮合规则是什么？"
- "融资开仓流程是怎样的？"
- "如何调用接口？"
- "账户开通需要什么条件？"
- "agw节点号在哪看？"
- "保证金不足怎么办？"

## 脚本参数

| 参数 | 说明 |
|------|------|
| `--question` | 用户问题（从用户消息自动提取） |
| `--system` | 系统分类：`低延时`、`顶点`、`all`（默认） |
| `--max-results` | 最大检索文档数（默认 5） |
| `--api-key` | MiniMax API Key（也可通过环境变量 `MINIMAX_API_KEY` 设置） |

## 输出格式

脚本返回格式化的文本答案，包含：
- 答案内容
- 参考文档链接
